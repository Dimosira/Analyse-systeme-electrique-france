# Analyse du système électrique français (RTE éCO2mix)

## Contexte
Ce projet vise à analyser les dynamiques de production et de consommation d’électricité en France à partir des données RTE (éCO2mix).

## Objectifs
- Identifier les déséquilibres entre production et consommation
- Détecter les périodes de tension énergétique
- Analyser les variations temporelles (saisonnalité, pics)

## Données
- Source : RTE éCO2mix
- Données spatio-temporelles (2013–2024)

##  Méthodologie
- Nettoyage et traitement des données
- Analyse exploratoire (EDA)
- Construction d’indicateurs (production, consommation, déficit)
- Analyse des tendances temporelles

##  Résultats
- Identification des pics de consommation (matin et hiver)
- Mise en évidence de tensions énergétiques (~24% en hiver)
- Analyse des disparités régionales

##  Outils
- Python (pandas, numpy)
- Power BI
- Data visualisation

##  Perspectives
- Amélioration du modèle avec des données climatiques
- Analyse prédictive des pics de consommation
